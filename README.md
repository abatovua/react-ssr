**React Server Side Rendering.**

Project is self-training app and for now only "/admin" pages available.

Requirements:
  1. NodeJS 7.7.4+
  2. npm 4.x+
  
Development:
  1. clone repo
  2. npm install
  3. npm run build:dev
  4. npm run server:dev
  5. open http://localhost:4200/admin
  
Prod: 
1. npm start
2. open http://localhost:4200/admin
